Política de Privacidad

Última actualización: 06 de febrero de 2026

1. Datos recopilados

Podemos recopilar información como nombre, número de teléfono, mensajes y datos necesarios para la operación del servicio.

2. Uso de la información

Los datos se utilizan únicamente para brindar el servicio de atención, ventas y automatización solicitados por el usuario.

3. Compartición de datos

No compartimos información personal con terceros, excepto cuando sea necesario para la operación del servicio o por requerimientos legales.

4. Seguridad

Aplicamos medidas razonables para proteger la información del usuario.
